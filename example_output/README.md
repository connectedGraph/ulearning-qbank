# 输出文件说明 / Output Files Documentation

本目录包含工具生成的输出文件示例和说明。

This directory contains examples and documentation for the output files generated by the tool.

## 📁 输出文件 / Output Files

### 1. `papers_json/` 目录

**内容**：下载的原始试卷JSON文件

**文件命名**：`{paperId}.json`

**示例**：`3415820.json`, `3415821.json`, ...

**文件结构**：
```json
{
  "code": 1,
  "message": "成功",
  "result": {
    "score": 100.0,
    "part": [
      {
        "paperpartid": 7940420,
        "partname": "期末测试练习题",
        "children": [
          {
            "questionid": 16143101,
            "title": "题目内容",
            "typeTitle": "单选题",
            "type": 1,
            "score": 5.0,
            "item": [
              {"choiceId": 37172898, "title": "选项A"},
              {"choiceId": 37172899, "title": "选项B"}
            ],
            "correctreply": ""
          }
        ]
      }
    ]
  }
}
```

**用途**：便于程序处理和二次开发

**文件结构**：
```json
[
  {
    "questionid": 10001,
    "title": "解码后的题目内容",
    "typeTitle": "单选题",
    "type": 1,
    "score": 5.0,

      {"choiceId": 1001, "title": "选项A"},
      {"choiceId": 1002, "title": "选项B"},
      {"choiceId": 1003, "title": "选项C"},
      {"choiceId": 1004, "title": "选项D"}

    "correctreply": ""

```

## 📊 数据说明 / Data Notes

### 题型分类 / Question Types

- **type: 1** → 单选题 (Single Choice)
- **type: 2** → 多选题 (Multiple Choice)
- **type: 4** → 判断题 (True/False)

### HTML 解码 / HTML Decoding

所有导出的题目都已经过HTML实体解码：

- `&ldquo;` → "
- `&rdquo;` → "
- `&ensp;` → (半角空格)
- `&nbsp;` → (不间断空格)

### 参考答案 / Reference Answers

- `correctreply` 字段通常为空字符串
- 服务器通常不下发答案解析
- 在Markdown中显示为"暂无（服务器未下发此题答案解析）"

## 📈 统计信息 / Statistics

运行 `analyzer.py` 后，会在控制台输出统计信息：

- 题目总数
- 题型分布
- 题目频次分布（出现1次、2次、...的题目数量）

示例输出：
```
题目频次分布 / Question Frequency Distribution:
  出现 1 次: 96 道题
  出现 2 次: 147 道题
  出现 3 次: 127 道题
  出现 4 次: 89 道题
  ...
```

---

**使用指南**：

1. 运行 `python src/downloader.py` 生成 `papers_json/` 目录
2. 运行 `python src/analyzer.py` 生成 `question_bank.json` 和 `question_bank.md`
3. 查看 `question_bank.md` 进行复习
4. 或使用 `question_bank.json` 进行二次开发（如制作刷题应用）
